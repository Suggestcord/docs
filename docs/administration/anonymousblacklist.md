# AnonymousBlacklist
---
### Description
Change whether the moderator's information is shown in suggestion logs when a user gets blacklisted.
### Arguments
`on | off` - Enable or disable the setting. - **Required**
### Usage
```
-anonymousblacklist <on/off>
```
### Aliases
`anonymous-blacklist`
### Cooldown
`2 seconds`
### Required Permissions
**Administrator**
