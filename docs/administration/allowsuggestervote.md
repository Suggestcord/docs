# AllowSuggesterVote
---
### Description
Change whether or not the suggester is allowed to vote with the reactions.
### Arguments
`on | off` - Enable or disable the setting. - **Required**
### Usage
```
-allowsuggestervote <on/off>
```
### Aliases
`allow-suggester-vote`
### Cooldown
`2 seconds`
### Required Permissions
**Administrator**
