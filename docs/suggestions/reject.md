# Reject
---
### Description
Reject a suggestion.
### Arguments
`sID` - ID of the suggestion. - **Required**\
`reason` - A reason to add to the rejected suggestion. - **Optional**
### Usage
```
-reject <sID> [reason]
```
### Cooldown
`2 seconds`
### Required Permissions
**Administrator** or **[Suggestions Manager](administration/suggestionsmanager.md)**
